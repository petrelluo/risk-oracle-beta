7.1 Defining and Using Sequences and Series

Sequences

A sequence is an ordered list of numbers called terms: a₁, a₂, a₃, …, aₙ.

A finite sequence has a last term. An infinite sequence continues without end.

A rule (explicit formula) gives aₙ directly as a function of n.

aₙ = some expression in n

Series & Summation Notation

A series is the sum of terms of a sequence.

Σᵢ₌₁ⁿ aᵢ = a₁ + a₂ + ⋯ + aₙ

i = index of summation; 1 = lower limit; n = upper limit

Special Summation Formulas

Σᵢ₌₁ⁿ 1 = n
Σᵢ₌₁ⁿ i = n(n+1)/2
Σᵢ₌₁ⁿ i² = n(n+1)(2n+1)/6

Examples

Ex 1 — Write first 5 terms of aₙ = 3n − 1

a₁=2, a₂=5, a₃=8, a₄=11, a₅=14

Sequence: 2, 5, 8, 11, 14, …

Ex 2 — Find a rule for: 2, 5, 10, 17, 26, …

2=1²+1, 5=2²+1, 10=3²+1 → pattern: n²+1

Rule: aₙ = n² + 1

Ex 3 — Write in sigma notation: 3+6+9+12+15

Each term = 3i for i = 1 to 5

Answer: Σᵢ₌₁⁵ 3i

Ex 4 — Evaluate Σᵢ₌₁⁴ (2i² − 1)

i=1: 1  |  i=2: 7  |  i=3: 17  |  i=4: 31

Sum = 56

7.2 Analyzing Arithmetic Sequences and Series

Arithmetic Sequence

Add constant d (common difference) each step.

aₙ = a₁ + (n − 1)d

d = a₂ − a₁ = a₃ − a₂ = … (constant)

Two terms: d = (aₙ − aₘ) / (n − m)

Arithmetic Series

Sₙ = n(a₁ + aₙ) / 2
Sₙ = n[2a₁ + (n−1)d] / 2

⚠ Need n, a₁, and aₙ or d.

Examples

Ex 5 — Rule & 20th term: 7, 11, 15, 19, …

d=4, a₁=7; aₙ = 7+(n−1)(4) = 4n+3

a₂₀ = 4(20)+3 = 83

Ex 6 — a₅ = 17 and a₁₂ = 45. Find a₁ and d.

7d = 45−17 = 28 → d = 4

17 = a₁+4(4) → a₁ = 1

Ex 7 — Find S₁₅ for 4, 9, 14, 19, …

a₁=4, d=5; a₁₅=4+14(5)=74

S₁₅ = 15(4+74)/2 = 585

Ex 8 — Evaluate Σᵢ₌₁²⁰ (3i − 2)

a₁=1, a₂₀=58; S₂₀=20(1+58)/2 = 590

Connection: Arithmetic sequence ↔ linear function; slope = d.

7.3 Analyzing Geometric Sequences and Series

Geometric Sequence

Multiply by constant r (common ratio) each step.

aₙ = a₁ · r^(n−1)

r = a₂/a₁ = a₃/a₂ = … (constant)

Two terms: r^(n−m) = aₙ/aₘ

Geometric Series

Sₙ = a₁(1 − rⁿ) / (1 − r),   r ≠ 1

If r = 1: Sₙ = n·a₁

Connection: Geometric sequence ↔ exponential function.

Examples

Ex 9 — Rule & 8th term: 3, 6, 12, 24, …

r=2, a₁=3; aₙ = 3·2^(n−1)

a₈ = 3·2⁷ = 384

Ex 10 — a₃ = 12, a₆ = 96. Find a₁ and r.

r³ = 96/12 = 8 → r=2; 12=a₁·4 → a₁=3

Ex 11 — S₆ for 2, −6, 18, −54, …

a₁=2, r=−3

S₆ = 2(1−(−3)⁶)/(4) = −364

Ex 12 — Evaluate Σᵢ₌₁⁵ 4·(1/2)^(i−1)

a₁=4, r=1/2, n=5

S₅ = 4(1−1/32)/(1/2) = 7.75

7.4 Finding Sums of Infinite Geometric Series

Convergence & Sum

Converges when |r| < 1; diverges when |r| ≥ 1.

S∞ = a₁ / (1 − r),   |r| < 1

Repeating Decimals

Express as geometric series to convert to a fraction.

0.333… = (3/10)/(1 − 1/10) = 3/9 = 1/3

0.1818… = (18/100)/(1 − 1/100) = 18/99 = 2/11

Examples

Ex 13 — Sum: 8 + 4 + 2 + 1 + …

r=1/2; S = 8/(1/2) = 16

Ex 14 — Σᵢ₌₁^∞ 3·(−2/5)^(i−1)

a₁=3, r=−2/5; S = 3/(7/5) = 15/7

Ex 15 — Write 0.1818… as a fraction

a₁=18/100, r=1/100

S = (18/100)/(99/100) = 2/11

Ex 16 — Does 1 + 1.1 + 1.21 + … converge?

r=1.1; |r|>1 → Diverges

7.5 Using Recursive Rules with Sequences

Recursive Rules

Defines each term using previous term(s) and a starting value.

Arithmetic:   a₁ = c,   aₙ = aₙ₋₁ + d
Geometric:   a₁ = c,   aₙ = r · aₙ₋₁

Converting Between Forms

a₁=c, aₙ=aₙ₋₁+d  ⟹  aₙ = c+(n−1)d

a₁=c, aₙ=r·aₙ₋₁  ⟹  aₙ = c·rⁿ⁻¹

Fibonacci-type: aₙ = aₙ₋₁ + aₙ₋₂ (needs a₁ and a₂)

Examples

Ex 17 — First 5 terms: a₁=4, aₙ=aₙ₋₁+7

4, 11, 18, 25, 32 — arithmetic, d=7

Explicit: aₙ = 7n − 3

Ex 18 — First 5 terms: a₁=2, aₙ=3aₙ₋₁

2, 6, 18, 54, 162 — geometric, r=3

Explicit: aₙ = 2·3^(n−1)

Ex 19 — Recursive rule for aₙ = 4n − 1

a₁=3, d=4 → a₁=3, aₙ=aₙ₋₁+4

Ex 20 — Recursive rule for aₙ = 5·(−2)^(n−1)

a₁=5, r=−2 → a₁=5, aₙ=−2·aₙ₋₁

Ex 21 — a₁=1, a₂=1, aₙ=aₙ₋₁+aₙ₋₂: first 6 terms

1, 1, 2, 3, 5, 8

Ex 22 — Savings: $100/mo, 0.5% interest/mo

aₙ = 1.005·aₙ₋₁ + 100, a₀=0

a₁=100, a₂=200.50, a₃≈301.50
