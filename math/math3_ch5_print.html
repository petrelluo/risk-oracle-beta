5.1   The Natural Base e   (The Euler Number)

Exponent Rules Review

  • Product of powers: \(b^m \cdot b^n = b^{m+n}\)
  • Power of a power: \((a^m)^n = a^{mn}\)
  • Quotient of powers: \(\dfrac{a^m}{a^n} = a^{m-n}\)
  • Negative exponent: \(a^{-n} = \dfrac{1}{a^n}\),   \(\dfrac{1}{a^{-n}}=a^n\),   \(\!\left(\dfrac{a}{b}\right)^{\!-n}=\!\left(\dfrac{b}{a}\right)^n\)
  • Power of a product: \((abc)^n = a^n b^n c^n\)

Exponential Function

Definition
A function of the form \(y = b^x\), where \(b > 0\) and \(b \neq 1\).
  • If \(b > 1\): the function models exponential growth
  • If \(0 < b < 1\): the function models exponential decay
  • If \(b = 1\): \(y = 1^x = 1\) — constant, not exponential
  • If \(b < 0\): expressions like \(b^{1/2}\) are undefined, so \(b\) must be positive

The parent exponential function \(y = b^x\) has domain: all real numbers, range: \(y > 0\), and asymptote: \(y = 0\) (x-axis) — the line the graph approaches but never touches.

Sample values for the parent function \(y = 2^x\):

\(x\)−3−2−10123
\(2^x\)\(\frac{1}{8}\)\(\frac{1}{4}\)\(\frac{1}{2}\)1248

Euler's Number e

Definition
\[e = \lim_{x \to \infty}\!\left(1 + \frac{1}{x}\right)^x \approx 2.71828\ldots\] \(e\) is irrational — its decimal expansion never ends or repeats (like \(\sqrt{2},\,\sqrt{3},\,\pi\)). Use \(e \approx 2.72\) for calculations.
Example 1 — Simplify expressions with e
a. \(e^2 \cdot e^9\)
= e^(2+9)   [product of powers: add exponents]
\(= e^{11}\)

Compound Interest Formula
\[A = P\!\left(1 + \frac{r}{n}\right)^{nt}\]
\(A\)Account balance (ending balance)
\(P\)Principal (initial deposit)
\(r\)Annual interest rate (decimal form)
\(n\)Compound frequency (times per year)
\(t\)Number of years
Example 3 — $100 at 5% annual interest for 2 years, compounded:
a. Yearly \((n=1)\):
A = 100(1+0.05)² \(\approx\$110.25\)
b. Semi-annually \((n=2)\):
A = 100(1.025)⁴ \(\approx\$110.38\)
c. Quarterly \((n=4)\):
A = 100(1+0.05/4)⁸ \(\approx\$110.45\)
d. Monthly \((n=12)\):
A = 100(1+0.05/12)^24 \(\approx\$110.49\)
e. Daily \((n=365)\):

\[A = Pe^{rt}\] This is the limit of \(P(1+r/n)^{nt}\) as \(n\to\infty\), where \(e\approx 2.72\) arises naturally.

5.2   Logarithms and Logarithmic Functions

Definition of Logarithm

The exponential equation \(2^3=8\) can be rewritten in logarithmic form as \(\log_2 8=3\) — read "log base 2 of 8."

Equivalent Forms
\[\log_b y = x \quad\Longleftrightarrow\quad b^x = y\]

Requires \(b>0,\ b\neq1,\ y>0\).

Example 1 — Rewrite each equation in exponential form
a. \(\log_6 216=3\) → \(6^3=216\)
b. \(\log_7 1=0\) → \(7^0=1\)
c. \(\log_{21}21=1\) → \(21^1=21\)
d. \(\log_{1/5}5=-1\) → \((1/5)^{-1}=5\)
Example 2 — Rewrite each expression in logarithmic form
a. \(9^2=81\) → \(\log_9 81=2\)
b. \(20^{-1}=0.05\) → \(\log_{20}0.05=-1\)
c. \(16^{3/4}=8\) → \(\log_{16}8=\tfrac{3}{4}\)
Example 3 — Evaluate each logarithm
a. \(\log_4 256\) — ask: \(4^?=256\). Since \(4^4=256\): \(=4\)
b. \(\log_{1/3}27\) — ask: \((1/3)^?=27\). Since \((1/3)^{-3}=3^3=27\): \(=-3\)
c. \(\log_8 0.125 = \log_8(1/8)\) — since \(8^{-1}=1/8\): \(=-1\)
d. \(\log_{64}8\) — since \(64^{1/2}=8\): \(=\tfrac{1}{2}\)

Common Log and Natural Log

Notation
  • \(\log x = \log_{10} x\)   — base 10, called the common logarithm
  • \(\ln x = \log_e x\)   — base \(e\), called the natural logarithm
Example 4 — Evaluate using a calculator (round to 3 decimal places)
a. \(\log 21 \approx\) 1.322
b. \(\ln 0.54 \approx\) −0.616

Special Values and Inverse Properties

Key Properties
  • \(\log_b b = 1\)   because \(b^1=b\)
  • \(\log_b 1 = 0\)   because \(b^0=1\)
  • \(\log_b b^x = x\)   (inverse property)
  • \(b^{\log_b x} = x\)   for \(x>0\)   (inverse property)

\(\log_b\)(\text{negative number}) is always undefined. For example, \(\log_{10}(-2)\) has no solution because \(10^?=-2\) is impossible.

Example 5 — Simplify using inverse properties
a. \(e^{\ln 18} = e^{\log_e 18} =\) 18
b. \(e^{\ln(-18)}\) → undefined (argument of ln must be positive)
c. \(\log_4 16^x = \log_4(4^2)^x = \log_4 4^{2x} =\) \(2x\)

Finding the Inverse of an Exponential or Logarithmic Function

To find the inverse: swap \(x\) and \(y\), then solve for \(y\). The inverse of an exponential function is logarithmic, and vice versa.

Example 6 — Find the inverse of each function
a. \(y=4^x\)
swap: x=4^y ↔ log₄x=y
\(y=\log_4 x\)
b. \(f(x)=11^x\)
x=11^y ↔ log₁₁x=y
\(f^{-1}=\log_{11}x\)
c. \(y=\ln(x-5)\)
x=ln(y−5) ↔ e^x=y−5
\(y=e^x+5\)
d. \(y=\log(x+6)\)
x=log(y+6) ↔ 10^x=y+6
\(y=10^x-6\)
e. \(y=2^x-9\)
x+9=2^y ↔ log₂(x+9)=y
\(y=\log_2(x+9)\)
f. \(y=-2+\log_3 x\)
x+2=log₃y ↔ 3^(x+2)=y
\(y=3^{x+2}\)
Example 7 — Find the inverse and graph both functions
a. \(f(x)=2^x\)
Take log₂ of both sides after swapping: log₂x = y
\(f^{-1}(x)=\log_2 x\)
b. \(f(x)=\log x\)

\(f^{-1}(x)=10^x\)

5.3   Transformations of Exponential and Logarithmic Functions

Parent Functions

Exponential: \(y=b^x\),  \(b>0,\,b\neq1\)
① \(b>1\) (e.g. \(b=2\) or \(e\)):
  • Domain: all reals  ·  Range: \(y>0\)
  • Asymptote: \(y=0\) — graph rises L to R
② \(0 (e.g. \(b=\tfrac{1}{2}\)):
  • Domain: all reals  ·  Range: \(y>0\)
  • Asymptote: \(y=0\) — graph falls L to R
Logarithmic: \(y=\log_b x\),  \(b>0,\,b\neq1\)
① \(b>1\) (e.g. \(y=\log_2 x\)):
  • Domain: \(x>0\)  ·  Range: all reals
  • Asymptote: \(x=0\) — graph rises L to R
② \(0 (e.g. \(y=\log_{1/2}x\)):
  • Domain: \(x>0\)  ·  Range: all reals
  • Asymptote: \(x=0\) — graph falls L to R

Transformation Rules

  • Vertical shift: \(g=f(x)+k\) — up \(k\) units (\(k>0\)) or down (\(k<0\)); asymptote shifts by \(k\)
  • Horizontal shift: \(g=f(x-h)\) — right \(h\) units (\(h>0\)) or left (\(h<0\)); asymptote shifts by \(h\)
  • Vertical stretch/shrink: \(g=a\cdot f(x)\) — stretches by \(|a|\); reflects over x-axis if \(a<0\)
  • Horizontal stretch/shrink: \(g=f(bx)\) — shrinks by \(|b|\); reflects over y-axis if \(b<0\)

Transformations also shift the domain, range, and asymptote accordingly.

Example 1 — Describe the transformation and graph: \(f(x)=(\frac{1}{3})^x\), \(g(x)=(\frac{1}{3})^x-5\)
\(g = f - 5\) → translate \(f\) down 5 units.
Asymptote: \(y=0\to y=-5\).   Range: \(y>-5\).
\(x\)−101
\(f=(1/3)^x\)31\(\frac{1}{3}\)
\(g=f-5\)−2−4\(-4\frac{2}{3}\)
Example 2 — Describe the transformation and graph: \(f(x)=e^x\), \(g(x)=e^{x+2}-1\)
Step 1: e^x → left 2 units → h(x)=e^(x+2)
Step 2: e^(x+2) → down 1 unit → g(x)=e^(x+2)−1
Asymptote: \(y=0\to y=-1\).   Range: \(y>-1\).
\(x\)−101
\(f=e^x\)\(\approx0.4\)1\(\approx2.7\)
Example 3 — Transformations of logarithmic functions: \(f(x)=\log x\), \(g(x)=\log(-\tfrac{1}{4}x)\)
Step 1: log x →[H stretch by 4]→ log(x/4) = log(¼x)
Step 2: log(¼x) →[reflect over y-axis]→ log(¼·(−x)) = log(−¼x)
Domain of \(g\): need \(-\tfrac{1}{4}x>0 \Rightarrow x<0\).   Asymptote: \(x=0\).
\(x\)−40−4−0.4
\(\log(-\tfrac{1}{4}x)\)10−1
Example 4 — Horizontal shift: \(f(x)=\log_{1/3}x\), \(g(x)=\log_{1/3}(x-2)\)
\(g\) = \(f\) shifted right 2 units.
Function \(f\): Domain \(x>0\), Asymptote \(x=0\)
\(x\)\(\frac{1}{3}\)13
\(y\)10−1
Function \(g\): Domain \(x>2\), Asymptote \(x=2\)
\(x\)\(2\tfrac{1}{3}\)35
\(y\)10−1
Example 5 — Write a rule for a composite transformation

Let \(g\) be: translate 1 unit down, then stretch vertically by factor 5, applied to \(f(x)=\log_{1/5}x\).

Step 1 (down 1): m = f−1 = (log₍₁/₅₎x)−1
Step 2 (stretch×5): g = 5·m = 5[(log₍₁/₅₎x)−1]
\(g = 5\log_{1/5}x - 5\)

5.4   Properties of Logarithms

Three Core Properties  (\(b>0,\,b\neq1,\,m>0,\,n>0\))
  1. Product Property:  \(\log_b(mn) = \log_b m + \log_b n\)
  2. Quotient Property:  \(\log_b\!\dfrac{m}{n} = \log_b m - \log_b n\)
  3. Power Property:  \(\log_b(m^n) = n\log_b m\)
Example 1 — Use \(\log_3 4\approx1.262\) and \(\log_3 5\approx1.465\) to evaluate
a. \(\log_3\tfrac{4}{5}\)
= log₃4 − log₃5 ≈ 1.262 − 1.465 = −0.203
b. \(\log_3 20\)
= log₃(4·5) = log₃4 + log₃5 ≈ 1.262 + 1.465 = 2.727
c. \(\log_3 25\) — two methods:
Method 1: log₃(5·5) = log₃5+log₃5 ≈ 1.465+1.465 = 2.93
Method 2: log₃5² = 2·log₃5 ≈ 2(1.465) = 2.93

Expanding a Logarithm

Expanding means rewriting a single logarithm as a sum or difference of simpler logarithms using the properties above.

Example 2 — Expand each logarithm
a. \(\log_2(3x^4)\)
= log₂3 + log₂x⁴  [product property]
\(= \log_2 3 + 4\log_2 x\)  [power property]
b. \(\ln\dfrac{3x^5}{y}\)
= ln(3x⁵) − ln y  [quotient property]
= ln3 + ln x⁵ − ln y  [product property]
\(= \ln3 + 5\ln x - \ln y\)  [power property]
c. \(\log_2\sqrt{x}\)
= log₂x^(1/2)   \(= \tfrac{1}{2}\log_2 x\)
d. \(\log\dfrac{5}{12x}\)
= log5 − log(12x)  [quotient property]
= log5 − (log12 + log x)  [product property]
\(= \log5 - \log12 - \log x\)

Condensing a Logarithm

Condensing means rewriting a sum or difference of logarithms as a single logarithm.

Coefficients must equal 1 before applying the product/quotient properties. Use the power property first to move coefficients up as exponents.
Example 3 — Condense each expression to a single logarithm
a. \(\log6 + 4\log3 - \log3\)
= log6 + 3log3  [combine: 4log3 − log3 = 3log3]
= 1·log6 + 1·log(3³)  [power property: coeff → exponent]
= log(6·27)  [product property]   \(= \log(162)\)
b. \(\ln4 - 3\ln3 - \ln12\)
= ln4 − ln(3³) − ln12  [power property]
= ln[ 4 / (27·12) ] = ln(4/324)   \(= \ln\dfrac{1}{3^4}\)

Alternate: treat negatives as exponents of −1:

= ln4 + ln(3⁻³) + ln(12⁻¹) = ln(4 · 3⁻³ · 12⁻¹) = ln(4/324)
c. \(-\log_2 x - \log_2 y - 3\)
Write −3 = −log₂8  [since log₂8=3]
= −(log₂x + log₂y + log₂8) = −log₂(8xy)
\(= \log_2\dfrac{1}{8xy}\)

Change-of-Base Formula

Allows any logarithm to be evaluated using a calculator's \(\log\) or \(\ln\) keys.

Formula
\[\log_b x = \frac{\log_c x}{\log_c b} = \frac{\log x}{\log b} = \frac{\ln x}{\ln b}\]
Example 4 — Evaluate using the Change-of-Base Formula
a. \(\log_2 5 = \dfrac{\log5}{\log2} \approx\) 2.322   (verify: \(\dfrac{\ln5}{\ln2} \approx 2.322\) ✓)
b. \(\log_{1/3}\dfrac{1}{9} = \dfrac{\log(1/9)}{\log(1/3)} =\) 2

5.5   Solving Exponential and Logarithmic Equations

Exponential equations are equations in which variable expressions occur as exponents.

Property of Equality for Exponential Equations
If \(b^x=b^y\) where \(b>0,\,b\neq1\), then \(x=y\).   Conversely, if \(x=y\), then \(b^x=b^y\).

Example: If \(3^m=3^n\), then \(m=n\).

Strategy 1 — Same Base: Rewrite and Equate Exponents

Example 1 — Solve by rewriting with the same base
a. \(81^x = \!\left(\tfrac{1}{9}\right)^{x+3}\)
(9²)^x = (9⁻¹)^(x+3)
9^(2x) = 9^(−(x+3))
2x = −(x+3)
2x = −x−3
3x = −3   \(x = -1\)
b. \(\!\left(\tfrac{1}{10}\right)^{-x} = 100^{3x+2}\)
(10⁻¹)^(−x) = (10²)^(3x+2)
10^x = 10^(6x+4)
x = 6x+4  ⟹  −4 = 5x   \(x = -\tfrac{4}{5}\)

Strategy 2 — Different Bases: Take a Logarithm of Both Sides

Example 2 — Solve by taking logarithms of both sides
a. \(3^x = 8\)
log₃(3^x) = log₃8
x = log₃8   \(\approx 1.893\)
b. \(7^{9x} = 15\)
log₇(7^(9x)) = log₇15
9x = log₇15  ⟹  x = log₇15 / 9   \(\approx 0.155\)
c. \(4e^{-0.3x} - 7 = 13\)
4e^(−0.3x) = 20
e^(−0.3x) = 5
ln(e^(−0.3x)) = ln5
−0.3x = ln5  ⟹  x = ln5/(−0.3)   \(\approx -5.365\)

Newton's Law of Cooling

Formula
\[T = (T_0 - T_R)\,e^{-rt} + T_R\]
\(T\)Final (current) temperature
\(T_0\)Initial temperature
\(T_R\)Surrounding (room) temperature
\(r\)Cooling rate
\(t\)Time (in minutes)
Example 3 — Newton's Law of Cooling

Chili is removed from the stove at 212°F. Room temperature is 75°F; cooling rate is 0.051. How long to cool to 95°F?

95 = (212−75)·e^(−0.051t) + 75
20 = 137·e^(−0.051t)
20/137 = e^(−0.051t)
ln(20/137) = −0.051t
t = ln(20/137) / (−0.051)   \(\approx 38 \text{ minutes}\)
Example 4 — Solving a compound interest equation for time

Ming invests at 5% annual interest compounded 12 times per year. How long until the balance doubles?

2P = P(1 + 0.05/12)^(12t)
2 = (1 + 0.05/12)^(12t)
ln2 = 12t · ln(1 + 0.05/12)  [power property of logs]
t = ln2 / [12·ln(1+0.05/12)]   \(\approx 13.89 \text{ years}\)

Solving Logarithmic Equations

Property of Equality for Logarithmic Equations
\[\log_b x = \log_b y \quad\Longleftrightarrow\quad x = y\] Requires \(b>0,\,b\neq1,\,x>0,\,y>0\).
Always check solutions. The argument of every logarithm must be positive — reject any answer that makes an argument zero or negative.

Two Methods for Logarithmic Equations

  • Method 1 (Same-base): Rewrite the constant as a log with the same base; equate arguments.
  • Method 2 (Exponentiate): Raise the base to both sides to convert to exponential form.
Example 5 — Solve logarithmic equations
a. \(\ln(5x+2) = \ln(x+10)\)
5x+2 = x+10  [property of equality]
4x = 8   \(x = 2\)
Check: ln(12) = ln(12) ✓
b. \(\log_3(10x-9) = 4\)
Method 1: log₃(10x−9) = log₃(3⁴) = log₃81
10x−9 = 81  ⟹  10x = 90   \(x = 9\)
Method 2: 3⁴ = 10x−9  ⟹  81 = 10x−9   \(x = 9\)
Example 6 — Condense first, then solve: \(\log_6 6x + \log_6(x-1) = 2\)
log₆[6x(x−1)] = 2  [product property]
6x(x−1) = 6² = 36  [exponentiate both sides]
x(x−1) = 6  ⟹  x²−x−6 = 0
(x−3)(x+2) = 0  ⟹  x=3 or x=−2
Reject \(x=-2\): \(\log_6(-12)\) is undefined.   \(x = 3\)

Solving Exponential and Logarithmic Inequalities

Solve the related equation to find the boundary point, then test a value on each side to determine the solution interval.

When \(0<b<1\), the function \(\log_b x\) is decreasing, so the inequality sign reverses when you exponentiate both sides.
Example 7 — Exponential inequality: \(3^x < 20\)
Boundary: 3^x = 20  ⟹  x = log₃20 ≈ 2.7
Test x=0: 3⁰ = 1 < 20 ✓ (true, so the interval x < 2.7 is correct)
\(x < 2.7\)
Example 8 — Logarithmic inequality: \(\log_{1/2} x \geq 5\)
Boundary: log_(1/2)x = 5  ⟹  (1/2)^5 = x  ⟹  x = 1/32
Base 1/2 < 1 → function is decreasing → inequality sign reverses when exponentiating.
Test x=1: log_(1/2)1 = 0 ≥ 5? → false → solution is on the side of x = 1/32 closer to 0.
\(0 < x \leq \dfrac{1}{32}\)

5.6   Modeling with Exponential and Logarithmic Equations

Identifying the Type of Function from a Table

Given equally spaced \(x\)-values, examine the \(y\)-values to determine the function type:

Classification Rules
  • Exponential: consecutive \(y\)-values share a constant ratio (multiply by the same factor each step)
  • Linear: 1st differences of \(y\) are constant
  • Quadratic: 2nd differences of \(y\) are constant
  • Cubic / Quartic: 3rd / 4th differences of \(y\) are constant

a.

\(y\)2438127931
Ratios: 243÷81=3, 81÷27=3, … constant ratio 1/3 as x increases by 1.
→ Exponential. At x=0, y=27:   \(y = 27\cdot\!\left(\tfrac{1}{3}\right)^x\)
b.
\(x\)−202468
\(y\)135133777133
1st differences: −8, 8, 24, 40, 56  (not constant)
2nd differences: 16, 16, 16, 16  (constant!) → Quadratic function (degree 2)

Writing an Exponential Function Through Two Points

Example 2 — Write \(y = a\cdot b^x\) whose graph passes through \((1,\,10)\) and \((3,\,40)\)
Substitute both points:
(1): 10 = a·b¹
(2): 40 = a·b³
Divide (2)÷(1) to eliminate a:
40/10 = b³/b = b²  ⟹  b² = 4  ⟹  b = ±2
Since b is a base and must satisfy b>0, b≠1:   b = 2
From (1): 10 = a·2  ⟹  a = 5
\(y = 5\cdot 2^x\)
