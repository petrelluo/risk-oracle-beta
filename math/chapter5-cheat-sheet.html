2 — Newton's Law of Cooling
T = (T₀ − TR) · e−rt + TR
T = final  |  T₀ = initial  |  TR = room temp  |  r = cooling rate  |  t = time
⚠ Common Mistakes
  • Don't confuse TR (room temp) with T (target temp).
  • Subtract TR from both sides before taking ln.
  • r is positive — the negative sign is already in the exponent.
Worked Example
Chili: T₀=212°F, TR=75°F, r=0.051. Find t when T=95°F:
95 = (212−75)·e−0.051t + 75
20/137 = e−0.051t
ln(20/137) = −0.051t
t = ln(20/137) ÷ (−0.051) ≈ 38 min
3 — Exponential & Logarithmic Inequalities
★ When Does the Inequality Sign Flip?
Multiply/Divide by Negative
−2x < 6  →  x > −3
Always flip. No exceptions.
Base 0 < b < 1 (Decreasing)
log½x ≥ 5  →  x ≤ (½)⁵
Flip — function is decreasing.
Base b > 1 (Increasing)
log3x ≥ 5  →  x ≥ 3⁵
No flip — function is increasing.
Exponential Inequalities — Examples
Base > 1 — No Flip
3x < 20  →  x < log320 ≈ 2.7
Test x=0: 1 < 20 ✓   Answer: x < 2.7
Base < 1 — Flip
(1/3)n ≥ 15
n ≤ log1/315 = ln15/ln(1/3) ≈ −2.46
Answer: n ≤ −2.46
Logarithmic Inequalities — Examples
State domain (argument > 0) first. Intersect with domain at the end.
Base < 1 — Flip
log½x ≥ 5   Domain: x > 0
x ≤ (½)⁵ = 1/32
Test x=1: log½1 = 0 ≥ 5? FALSE ✓
Answer: 0 < x ≤ 1/32
Condense First, Base > 1 — No Flip
log2(x+1)+log2(x−1) > 3   Domain: x > 1
log2(x²−1) > 3  →  x²−1 > 8  →  x² > 9
x > 3 or x < −3, intersect with x > 1:
Answer: x > 3
