Sections 5.1 – 5.6 · Exponential & Logarithmic Functions

5.1

The Natural Base e

The Number e
e ≈ 2.71828…

Irrational constant. The natural base of exponential functions.

Natural Exponential Function
f(x) = ex

Domain: all reals. Range: y > 0. Passes through (0, 1) and (1, e). Always increasing.

Continuous Compounding
A = Pert

P = principal, r = annual rate (decimal), t = years.

Inverse Properties
eln x = x  &  ln(ex) = x

y = ex and y = ln x are inverses of each other.

💡

The graph of f(x) = ex has a horizontal asymptote at y = 0. It is strictly increasing for all x and is never negative.

Example — Continuous Compounding

$3,000 invested at 4% compounded continuously for 6 years:
A = 3000 · e0.04 · 6 = 3000 · e0.24 ≈ 3000 × 1.2712 ≈ $3,813.57

5.2

Logarithms and Logarithmic Functions

Definition
logb y = x  ⟺  bx = y

Requires b > 0, b ≠ 1, and y > 0.

Common Log
log x = log10 x

Base 10. Used when no base is written. log 100 = 2 since 10² = 100.

Natural Log
ln x = loge x

Base e. Inverse of y = ex. ln e = 1.

Special Values
logb 1 = 0
logb b = 1
logb(bx) = x

These follow directly from the definition.

📌

Graph of y = logb x: domain x > 0, range all reals, vertical asymptote x = 0, passes through (1, 0) and (b, 1). It is the reflection of y = bx over the line y = x.

5.4

Properties of Logarithms

Product Property
logb(mn) = logb m + logb n

Log of a product equals the sum of logs.

Quotient Property
logb mn = logb m − logb n

Log of a quotient equals the difference of logs.

Power Property
logb(mn) = n · logb m

The exponent moves to become a multiplier in front.

Change of Base
logb a = log alog b = ln aln b

Converts any base to base 10 or base e for calculator use.

Expand

log2 8x3y = log2 8 + 3 log2 x − log2 y = 3 + 3 log2 x − log2 y

Condense

2 ln x + ln 5 − ln y = ln(x2) + ln 5 − ln y = ln 5x2y

5.5

Solving Exponential & Logarithmic Equations

Exponential Equations

  1. Same base: If bu = bv, then u = v. (Set exponents equal.)
  2. Different bases: Take log or ln of both sides → use Power Property to bring variable down → solve.
  3. With e: Isolate eu, then apply ln to both sides.
Example — Different Bases

Solve 32x = 50:
2x · ln 3 = ln 50  →  2x = ln 50ln 3  →  x = ln 502 ln 3 ≈ 1.786

Logarithmic Equations

  1. Condense both sides to a single logarithm each.
  2. Convert to exponential form (or if logb u = logb v, then u = v).
  3. Solve the resulting equation.
  4. Check: Reject any solution that makes a log argument ≤ 0 (extraneous root).
Example — Logarithmic with Extraneous Root

Solve log3(x + 4) + log3(x − 2) = 3:
log3[(x+4)(x−2)] = 3  →  (x+4)(x−2) = 27
x2 + 2x − 8 = 27  →  x2 + 2x − 35 = 0  →  (x+7)(x−5) = 0
x = −7: log of negative → reject.    x = 5 ✓   Answer: x = 5

⚠️

Always check for extraneous solutions when solving logarithmic equations. Any value that makes a logarithm's argument ≤ 0 must be discarded.

Finding a Model from Two Points
Use y = a · bx

Plug in both points to get two equations. Divide to eliminate a, solve for b, then find a.

Finding Rate from Growth Data
Solve aekt = y for k

Isolate ekt, apply ln, divide by t. Then use k to find any other value.

Carbon-14 half-life: 5,730 years. Starting with 200 g, find amount after 10,000 yr:
y = 200 · 1210000/5730 = 200 · (0.5)1.746 ≈ 59.5 g
