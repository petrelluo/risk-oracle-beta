5.1   The Natural Base e  (The Euler Number)

Exponent Rules Review

Before working with exponential functions, recall these key properties of exponents:

  • Product of powers: \(b^m \cdot b^n = b^{m+n}\)   e.g. \(2^3 \cdot 2^5 = 2^8\)
  • Power of a power: \((a^m)^n = a^{mn}\)   e.g. \((a^3)^4 = a^{12}\)
  • Quotient of powers: \(\dfrac{a^m}{a^n} = a^{m-n}\)   e.g. \(\dfrac{a^3}{a^7} = a^{-4}\)
  • Negative exponent: \(a^{-4} = \dfrac{1}{a^4}\),   \(\dfrac{1}{a^{-4}} = a^4\),   \(\!\left(\dfrac{a}{b}\right)^{\!-4} = \!\left(\dfrac{b}{a}\right)^{\!4}\)
  • Power of a product: \((a \cdot b \cdot c)^3 = a^3 b^3 c^3\)

Exponential Function

Definition
A function of the form \(y = b^x\), where \(b > 0\) and \(b \neq 1\).
  • If \(b > 1\): the function models exponential growth
  • If \(0 < b < 1\): the function models exponential decay
  • If \(b = 1\): \(y = 1^x = 1\) (constant — not an exponential function)
  • If \(b < 0\) (e.g. \(b = -2\)): expressions like \((-2)^{1/2} = \sqrt{-2}\) are undefined, so \(b\) must be positive

The parent exponential function \(y = b^x\) has:

  • Domain: all real numbers (ARN)
  • Range: \(y > 0\)
  • Asymptote: \(y = 0\) (the x-axis) — the line the graph approaches but never touches

Sample values for \(y = 2^x\):

\(x\)−3−2−10123
\(y = 2^x\)\(\tfrac{1}{8}\)\(\tfrac{1}{4}\)\(\tfrac{1}{2}\)1248

Euler's Number e

Definition
\[e = \lim_{x \to \infty}\!\left(1 + \frac{1}{x}\right)^x \approx 2.71828\ldots\] \(e\) is irrational — its decimal expansion never ends or repeats (like \(\sqrt{2},\, \sqrt{3},\, \pi\)). For calculations, use \(e \approx 2.72\).

Simple Interest vs. Compound Interest

Simple interest: interest comes from the principal (initial deposit) only.

Example with \(P = \$100\), annual rate \(r = 8\%\):

YearBeginning BalanceEnding Balance \(A\)
1$100\(100 + 8\%(100) = 108\)
2$108\(108 + 8\%(100) = 116\)
3$116\(124\)
\(t\)—\(A = 100 + 8t\)   (linear growth)

Compound interest: interest comes from both the principal and the previously-earned interest.

\(n\) = compound frequency — how often the interest is calculated every year.

With \(n = 1\) (once a year), \(r = 8\%\):

YearBeginning BalanceEnding Balance \(A\)
1$100\(100 + 8\%(100) = 108\)
2$108\(108 + 8\%(108) = 116.64\)   ← 64¢ more than simple interest

With \(n = 2\) (twice a year), \(r = 8\%\):

YearBeginning BalanceEnding Balance
\(\tfrac{1}{2}\)$100\(100 + 4\%(100) = 104\)
1$104\(104 + 4\%(104) = 108.16\)
\(1\tfrac{1}{2}\)$108.16\(108.16 + 4\%(108.16) = 112.49\)
2$112.49\(112.49 + 4\%(112.49) \approx 116.99\)

Check: \(A = 100\!\left(1+\tfrac{8\%}{2}\right)^{2\cdot 2} = 100(1.04)^4 \approx 116.99\) ✓

Compound Interest Formula
\[A = P\!\left(1 + \frac{r}{n}\right)^{nt}\]
\(A\)Account balance (ending balance)
\(P\)Principal (initial deposit)
\(r\)Annual interest rate (as a decimal)
\(n\)Compound frequency (times per year)
\(t\)Number of years
Example 3  –  $100 deposited at 5% annual interest, find balance after 2 years compounded as:
a. Yearly (\(n=1\)):
\(A = 100\!\left(1+\tfrac{0.05}{1}\right)^{1\cdot 2} = 100(1.05)^2 \approx\) \(\$110.25\)
b. Semi-annually (\(n=2\)):
\(A = 100\!\left(1+\tfrac{0.05}{2}\right)^{2\cdot 2} = 100(1.025)^4 \approx\) \(\$110.38\)
c. Quarterly (\(n=4\)):
\(A = 100\!\left(1+\tfrac{0.05}{4}\right)^{4\cdot 2} \approx\) \(\$110.45\)
d. Monthly (\(n=12\)):
\(A = 100\!\left(1+\tfrac{0.05}{12}\right)^{12\cdot 2} \approx\) \(\$110.49\)
e. Daily (\(n=365\)):
\(A = 100\!\left(1+\tfrac{0.05}{365}\right)^{365\cdot 2} \approx\) \(\$110.52\)
f. Continuously (\(n \to \infty\)):
\(A = Pe^{rt} = 100\cdot e^{0.05\cdot 2} = 100e^{0.1} \approx\) \(\$110.52\)

Notice that as \(n\) increases, the balance grows but levels off. When \(n \to \infty\), we get the continuously compounded formula:

Continuously Compounded Interest
\[A = Pe^{rt}\] This is the limiting case of \(A = P\!\left(1+\frac{r}{n}\right)^{nt}\) as \(n \to \infty\), where the natural base \(e \approx 2.72\) arises naturally.

5.2   Logarithms and Logarithmic Functions

Definition of Logarithm

The exponential equation \(2^3 = 8\) can be rewritten in logarithmic form as \(\log_2 8 = 3\) — read "log base 2 of 8."

Equivalent Forms
\[\log_b y = x \quad \Longleftrightarrow \quad b^x = y\]
Requires \(b > 0,\ b \neq 1,\ y > 0\).
Example 1  –  Rewrite each equation in exponential form
a. \(\log_6 216 = 3\) → \(6^3 = 216\)
b. \(\log_7 1 = 0\) → \(7^0 = 1\)
c. \(\log_{21} 21 = 1\) → \(21^1 = 21\)
d. \(\log_{1/5} 5 = -1\) → \(\!\left(\tfrac{1}{5}\right)^{-1} = 5\)
Example 2  –  Rewrite each expression in logarithmic form
a. \(9^2 = 81\) → \(\log_9 81 = 2\)
b. \(20^{-1} = 0.05\) → \(\log_{20} 0.05 = -1\)
c. \(16^{3/4} = 8\) → \(\log_{16} 8 = \tfrac{3}{4}\)
Example 3  –  Evaluate each logarithm
a. \(\log_4 256 = ?\)   Ask: \(4^? = 256\). Since \(4^4 = 256\),   \(= 4\)
b. \(\log_{1/3} 27 = ?\)   Ask: \((1/3)^? = 27\). Since \((1/3)^{-3} = 3^3 = 27\),   \(= -3\)
c. \(\log_8 0.125 = \log_8 \tfrac{1}{8} = ?\)   Since \(8^{-1} = \tfrac{1}{8}\),   \(= -1\)
d. \(\log_{64} 8 = ?\)   Ask: \(64^? = 8\). Since \(64^{1/2} = 8\),   \(= \tfrac{1}{2}\)

Common Log and Natural Log

Notation
  • \(\log x = \log_{10} x\)   (base 10, called the common logarithm)
  • \(\ln x = \log_e x\)   (base \(e\), called the natural logarithm)
Example 4  –  Evaluate using a calculator (round to 3 decimal places)
a. \(\log 21 \approx\) 1.322
b. \(\ln 0.54 \approx\) −0.616

Special Logarithm Values and Inverse Properties

Key Properties
  • \(\log_b b = 1\)   because \(b^1 = b\)
  • \(\log_b 1 = 0\)   because \(b^0 = 1\)
  • \(\log_b b^x = x\)   (inverse property)
  • \(b^{\log_b x} = x\)   for \(x > 0\)   (inverse property)

Note: \(\log_b(\text{negative number})\) is undefined. For example, \(\log_{10}(-2)\) is undefined because \(10^? = -2\) has no real solution.

Example 5  –  Simplify using inverse properties
a. \(e^{\ln 18} = e^{\log_e 18} =\) 18
b. \(e^{\ln(-18)}\) → undefined   (argument of ln must be positive)
c. \(\log_4 16^x = \log_4(4^2)^x = \log_4 4^{2x} =\) \(2x\)

Finding the Inverse of an Exponential Function

To find the inverse: swap \(x\) and \(y\), then solve for \(y\). The inverse of an exponential function is a logarithmic function, and vice versa.

Example 6  –  Find the inverse of each function
a. \(y = 4^x\)
Swap: \(x = 4^y\)   (exponential form) \(\Longleftrightarrow\) \(\log_4 x = y\)   (log form)
Inverse: \(y = \log_4 x\)
b. \(f(x) = 11^x\)
Swap: \(x = 11^y \Longleftrightarrow \log_{11} x = y\)
Inverse: \(f^{-1}(x) = \log_{11} x\)
c. \(y = \ln(x-5)\)
Swap: \(x = \ln(y-5) \Longleftrightarrow e^x = y-5\)
Inverse: \(y = e^x + 5\)
d. \(y = \log(x+6)\)
Swap: \(x = \log(y+6) \Longleftrightarrow 10^x = y+6\)
Inverse: \(y = 10^x - 6\)
e. \(y = 2^x - 9\)
Swap: \(x = 2^y - 9 \Rightarrow x+9 = 2^y \Longleftrightarrow \log_2(x+9) = y\)
Inverse: \(y = \log_2(x+9)\)
f. \(y = -2 + \log_3 x\)
Swap: \(x = -2 + \log_3 y \Rightarrow x+2 = \log_3 y \Longleftrightarrow 3^{x+2} = y\)
Inverse: \(y = 3^{x+2}\)
Example 7  –  Find the inverse and graph both functions
a. \(f(x) = 2^x\)
Take \(\log_2\) of both sides: \(\log_2 x = \log_2 2^y = y\)
Inverse: \(f^{-1}(x) = \log_2 x\)
b. \(f(x) = \log x\)
Swap and exponentiate: \(10^x = 10^{\log_{10} y} = y\)
Inverse: \(f^{-1}(x) = 10^x\)

5.3   Transformations of Exponential and Logarithmic Functions

Parent Functions

Exponential Parent: \(y = b^x\),   \(b > 0,\, b \neq 1\)
Case ① \(b > 1\)  (e.g. \(b = 2\) or \(e\)):
  • Domain: ARN  ·  Range: \(y > 0\)
  • Asymptote: \(y = 0\) (x-axis)
  • Graph rises left to right (growth)
Case ② \(0 < b < 1\)  (e.g. \(b = \frac{1}{2}\)):
  • Domain: ARN  ·  Range: \(y > 0\)
  • Asymptote: \(y = 0\) (x-axis)
  • Graph falls left to right (decay)
Logarithmic Parent: \(y = \log_b x\),   \(b > 0,\, b \neq 1\)
Case ① \(b > 1\)  (e.g. \(y = \log_2 x\)):
  • Domain: \(x > 0\)  ·  Range: ARN
  • Asymptote: \(x = 0\) (y-axis)
  • Graph rises left to right
Case ② \(0 < b < 1\)  (e.g. \(y = \log_{1/2} x\)):
  • Domain: \(x > 0\)  ·  Range: ARN
  • Asymptote: \(x = 0\) (y-axis)
  • Graph falls left to right

Describing Transformations

To describe how \(g\) is a transformation of parent function \(f\), compare the equations and identify each change.

  • Vertical shift: \(g = f(x) + k\) — up \(k\) units (\(k > 0\)) or down (\(k < 0\)). Shifts the asymptote by \(k\).
  • Horizontal shift: \(g = f(x - h)\) — right \(h\) units (\(h > 0\)) or left (\(h < 0\)). Shifts the asymptote by \(h\).
  • Vertical stretch/shrink: \(g = a \cdot f(x)\) — stretches by \(|a|\); reflects over x-axis if \(a < 0\).
  • Horizontal stretch/shrink: \(g = f(b \cdot x)\) — shrinks by factor \(|b|\); reflects over y-axis if \(b < 0\).

Transformations also shift the domain, range, and asymptote accordingly.

Example 1  –  Describe the transformation of \(f\) represented by \(g\), then graph
Setup: \(f(x) = \!\left(\tfrac{1}{3}\right)^x\),   \(g(x) = \!\left(\tfrac{1}{3}\right)^x - 5\)
\(g = f - 5\)  → translate \(f\) down 5 units.
Asymptote shifts from \(y=0\) to \(y=-5\). Range: \(y > -5\).
\(x\)−101
\(f=(1/3)^x\)31\(\frac{1}{3}\)
\(g=f-5\)−2−4\(-4\tfrac{2}{3}\)

Setup: \(f(x) = e^x\),   \(g(x) = e^{x+2} - 1\)
Step 1: \(e^x \xrightarrow{\text{left 2 units}} h(x) = e^{x+2}\)
Step 2: \(e^{x+2} \xrightarrow{\text{down 1 unit}} g(x) = e^{x+2} - 1\)

Example 5  –  Write a rule for a composite transformation

Let \(g\) be: a translation 1 unit down followed by a vertical stretch by factor 5 of \(f(x) = \log_{1/5} x\). Write a rule for \(g\).

5.4   Properties of Logarithms

Three Core Properties   (\(b>0,\,b\neq1,\,m>0,\,n>0\))
  1. Product Property:  \(\log_b(mn) = \log_b m + \log_b n\)
  2. Quotient Property:  \(\log_b\!\dfrac{m}{n} = \log_b m - \log_b n\)
  3. Power Property:  \(\log_b(m^n) = n\log_b m\)
Example 1  –  Use \(\log_3 4 \approx 1.262\) and \(\log_3 5 \approx 1.465\) to evaluate
a. \(\log_3\!\dfrac{4}{5}\)
\(= \log_3 4 - \log_3 5 \approx 1.262 - 1.465 =\) \(-0.203\)
b. \(\log_3 20\)
\(= \log_3(4 \cdot 5) = \log_3 4 + \log_3 5 \approx 1.262 + 1.465 =\) \(2.727\)
c. \(\log_3 25\) — two methods:
Method 1: \(\log_3(5 \cdot 5) = \log_3 5 + \log_3 5 \approx 1.465 + 1.465 = 2.93\)
Method 2: \(\log_3 5^2 = 2\log_3 5 \approx 2(1.465) =\) \(2.93\)

Expanding a Logarithm

Expanding a logarithm means rewriting a single logarithm as a sum or difference of simpler logarithms using the properties above.

Example 2  –  Expand each logarithm
a. \(\log_2(3x^4)\)
\(= \log_2 3 + \log_2 x^4\)   (product property)
\(= \log_2 3 + 4\log_2 x\)   (power property)
b. \(\ln\!\dfrac{3x^5}{y}\)
\(= \ln 3x^5 - \ln y\)   (quotient property)
\(= \ln 3 + \ln x^5 - \ln y\)   (product property)
\(= \ln 3 + 5\ln x - \ln y\)   (power property)
c. \(\log_2 \sqrt{x}\)
\(= \log_2 x^{1/2}\) \(= \tfrac{1}{2}\log_2 x\)
d. \(\log\!\dfrac{5}{12x}\)
\(= \log 5 - \log 12x\)   (quotient property)
\(= \log 5 - (\log 12 + \log x)\)   (product property)
\(= \log 5 - \log 12 - \log x\)

Note: \(\dfrac{5}{12x}\) means \(5 \div 12 \div x\).

Condensing a Logarithm

Condensing means rewriting a sum or difference of logarithms as a single logarithm.

Coefficients must equal 1 before applying the product/quotient properties — use the power property first to move coefficients to exponents.
Example 3  –  Condense each expression to a single logarithm
a. \(\log 6 + 4\log 3 - \log 3\)
\(= \log 6 + 3\log 3\)   (combine like terms: \(4\log 3 - \log 3 = 3\log 3\))
\(= \log 6 + \log 3^3\)   (power property: coefficient → exponent)
\(= \log(6 \cdot 27)\)   (product property)
\(= \log(162)\)
b. \(\ln 4 - 3\ln 3 - \ln 12\)
\(= \ln 4 - \ln 3^3 - \ln 12\)   (power property)
\(= \ln\!\dfrac{4}{3^3 \cdot 12} = \ln\!\dfrac{4}{324} = \ln\!\dfrac{1}{81}\)
\(= \ln\!\dfrac{1}{3^4}\)

Alternate approach: treat all as positive with signs in exponents:

\(= \ln 4 + \ln 3^{-3} + \ln 12^{-1} = \ln\!\left(4 \cdot 3^{-3} \cdot 12^{-1}\right) = \ln\!\dfrac{4}{3^3 \cdot 12}\)
c. \(-\log_2 x - \log_2 y - 3\)
\(-3 = -\log_2 8\)   (since \(\log_2 8 = 3\))
\(= -\log_2 x - \log_2 y - \log_2 8\)
\(= -(\log_2 x + \log_2 y + \log_2 8)\)
\(= -\log_2(8xy)\)
\(= \log_2\!\dfrac{1}{8xy}\)

Change-of-Base Formula

Allows any logarithm to be evaluated using a calculator's \(\log\) or \(\ln\) keys.

Formula
\[\log_b x = \frac{\log_c x}{\log_c b} = \frac{\log x}{\log b} = \frac{\ln x}{\ln b}\]
Example 4  –  Evaluate using the Change-of-Base Formula
a. \(\log_2 5 = \dfrac{\log 5}{\log 2} \approx\) 2.322   (verify: \(\dfrac{\ln 5}{\ln 2} \approx 2.322\) ✓)
b. \(\log_{1/3}\!\dfrac{1}{9} = \dfrac{\log(1/9)}{\log(1/3)} =\) 2

5.5   Solving Exponential and Logarithmic Equations

Exponential equations are equations in which variable expressions occur as exponents.

Property of Equality for Exponential Equations
If \(b^x = b^y\) where \(b > 0,\, b \neq 1\), then \(x = y\).   Conversely, if \(x = y\), then \(b^x = b^y\).

Example: If \(3^m = 3^n\), then \(m = n\).

Example 1  –  Solve by rewriting both sides with the same base, then equating exponents
a. \(81^x = \!\left(\tfrac{1}{9}\right)^{x+3}\)
\((9^2)^x = (9^{-1})^{x+3}\)
\(9^{2x} = 9^{-(x+3)}\)
\(2x = -(x+3)\)
\(2x = -x - 3\)
\(3x = -3\)
\(x = -1\)
b. \(\!\left(\tfrac{1}{10}\right)^{-x} = 100^{3x+2}\)
\((10^{-1})^{-x} = (10^2)^{3x+2}\)
\(10^x = 10^{6x+4}\)
\(x = 6x + 4\)
\(-4 = 5x\)
\(x = -\dfrac{4}{5}\)
Example 2  –  Solve by taking a logarithm of both sides
Use this strategy when both sides cannot be written with the same base.
a. \(3^x = 8\)
\(\log_3 3^x = \log_3 8\)
\(x = \log_3 8\)   \(\approx 1.893\)
b. \(7^{9x} = 15\)
\(\log_7 7^{9x} = \log_7 15\)
\(9x = \log_7 15\)
\(x = \dfrac{\log_7 15}{9}\)   \(\approx 0.155\)
c. \(4e^{-0.3x} - 7 = 13\)
\(4e^{-0.3x} = 20\)
\(e^{-0.3x} = 5\)
\(\log_e e^{-0.3x} = \log_e 5\)
\(-0.3x = \ln 5\)
\(x = \dfrac{\ln 5}{-0.3}\)   \(\approx -5.365\)

Newton's Law of Cooling

Formula
\[T = (T_0 - T_R)\,e^{-rt} + T_R\]
\(T\)Final (current) temperature
\(T_0\)Initial temperature
\(T_R\)Surrounding (room) temperature
\(r\)Cooling rate
\(t\)Time in minutes
Example 3  –  Apply Newton's Law of Cooling

Your sister takes her chili off the stove at 212°F. The room temperature is 75°F and the cooling rate is 0.051. How long does it take to cool to a serving temperature of 95°F?

\(T = (T_0 - T_R)e^{-rt} + T_R\)
\(95 = (212 - 75)e^{-0.051t} + 75\)
\(95 - 75 = (212 - 75)e^{-0.051t}\)
\(\dfrac{95-75}{212-75} = e^{-0.051t}\)
\(\ln\!\dfrac{20}{137} = -0.051t\)
\(t = \dfrac{\ln(20/137)}{-0.051}\)
\(t \approx 38 \text{ minutes}\)
Example 4  –  Solving a compound interest equation for time

Ming invested money at 5% annual interest compounded 12 times a year. How long until the balance doubles?

\(A = P\!\left(1+\dfrac{r}{n}\right)^{nt}\)
\(2P = P\!\left(1+\dfrac{0.05}{12}\right)^{12t}\)
\(2 = \!\left(1+\dfrac{0.05}{12}\right)^{12t}\)
\(\ln 2 = \ln\!\left(1+\dfrac{0.05}{12}\right)^{12t}\)
\(\ln 2 = 12t \cdot \ln\!\left(1+\dfrac{0.05}{12}\right)\)
\(t = \dfrac{\ln 2}{12 \cdot \ln\!\left(1+\frac{0.05}{12}\right)}\)
\(t \approx 13.89 \text{ years}\)
Property of Equality for Logarithmic Equations
\[\log_b x = \log_b y \quad \Longleftrightarrow \quad x = y\] Requires \(b > 0,\, b \neq 1,\, x > 0,\, y > 0\).
Always check your solutions for logarithmic equations — the argument of a logarithm must be positive. Reject any solution that makes an argument zero or negative.

Two Methods for Solving Logarithmic Equations

  • Method 1 (Same-base): Rewrite the constant side as a logarithm with the same base, then equate the arguments.
  • Method 2 (Exponentiate): Raise the base to both sides to convert from logarithmic to exponential form.
Example 5  –  Solve logarithmic equations
a. \(\ln(5x+2) = \ln(x+10)\)
\(5x + 2 = x + 10\)   (property of equality)
\(4x = 8\)
\(x = 2\)
Check: \(\ln(5(2)+2) = \ln 12 = \ln(2+10)\) ✓
b. \(\log_3(10x-9) = 4\)
Method 1: \(\log_3(10x-9) = \log_3 81\)   (since \(3^4=81\))
\(10x - 9 = 81 \Rightarrow 10x = 90\)   \(x = 9\)

Method 2: \(3^4 = 10x - 9 \Rightarrow 81 = 10x - 9 \Rightarrow 10x = 90\)   \(x = 9\)
Example 6  –  Condense first, then solve
Setup: \(\log_6 6x + \log_6(x-1) = 2\)
\(\log_6\bigl[6x(x-1)\bigr] = 2\)   (product property)
\(6^2 = 6x(x-1)\)   (exponentiate both sides)
\(36 = 6x(x-1)\)
\(6 = x(x-1)\)
\(x^2 - x - 6 = 0\)
\((x-3)(x+2) = 0\)
\(x = 3\) or \(x = -2\)
Reject \(x = -2\) since \(\log_6(-12)\) is undefined.   \(x = 3\)

Solving Exponential and Logarithmic Inequalities

Solve the related equation to find the boundary point, then test a value on each side to determine which interval satisfies the inequality.

When \(0 < b < 1\), the function \(\log_b x\) is decreasing, so the inequality sign reverses when you exponentiate both sides.
Example 7  –  Solve an exponential inequality
Setup: \(3^x < 20\)
Boundary: \(3^x = 20 \Rightarrow x = \log_3 20 \approx 2.7\)
Test \(x = 0\): \(3^0 = 1 < 20\) ✓ (true, so the region \(x < 2.7\) satisfies the inequality)
\(x < 2.7\)
Example 8  –  Solve a logarithmic inequality
Setup: \(\log_{1/2} x \geq 5\)
Boundary: \(\log_{1/2} x = 5 \Rightarrow \!\left(\tfrac{1}{2}\right)^5 = x \Rightarrow x = \tfrac{1}{32}\)
Since base \(\tfrac{1}{2} < 1\), the function is decreasing.
Test \(x = 1\): \(\log_{1/2} 1 = 0 \geq 5\)?   False → the inequality holds on the side of \(x = \tfrac{1}{32}\) closer to 0.
\(0 < x \leq \dfrac{1}{32}\)

5.6   Modeling with Exponential and Logarithmic Equations

Identifying the Type of Function from a Table

Given equally spaced \(x\)-values, examine the \(y\)-values:

  • Exponential: consecutive \(y\)-values have a constant ratio (multiply by the same factor each step).
  • Linear: 1st differences of \(y\) are constant.
  • Quadratic: 2nd differences of \(y\) are constant.
  • Cubic / Quartic: 3rd / 4th differences of \(y\) are constant.
Example 1  –  Determine the type of function represented by each table
a.
\(x\)−2−10123
\(y\)2438127931
Each step: \(243 \div 3 = 81,\; 81 \div 3 = 27,\ldots\)   constant ratio \(\tfrac{1}{3}\) as \(x\) increases by 1.
→ Exponential function.   At \(x=0\), \(y=27\), so: \(y = 27\cdot\!\left(\tfrac{1}{3}\right)^x\)
b.
\(x\)−202468
\(y\)135133777133
\(x\)-values are equally spaced (by 2).
1st differences: \(-8,\ 8,\ 24,\ 40,\ 56\)   (not constant)
2nd differences: \(16,\ 16,\ 16,\ 16\)   (constant!)
→ Quadratic function (polynomial degree 2)

Writing an Exponential Function Through Two Points

Example 2  –  Write \(y = a \cdot b^x\) passing through \((1,\,10)\) and \((3,\,40)\)
Substitute both points:
\(\quad 10 = a \cdot b^1 \qquad (1)\)
\(\quad 40 = a \cdot b^3 \qquad (2)\)

Divide equation (2) by (1) to eliminate \(a\):
\(\dfrac{40}{10} = \dfrac{ab^3}{ab^1} \Rightarrow 4 = b^2 \Rightarrow b = \pm 2\)

Since \(b\) is the base and must satisfy \(b > 0,\, b \neq 1\):   \(b = 2\)

Substitute back into (1): \(10 = a \cdot 2 \Rightarrow a = 5\)
\(y = 5 \cdot 2^x\)
