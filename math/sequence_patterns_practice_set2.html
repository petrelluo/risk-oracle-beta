Strategy
How to use this set: These 10 problems use the same six pattern families from Set 1, but in different forms. The goal is automaticity — when Claire sees a sequence, she should reach for the checklist without thinking.

The Six-Step Checklist (same as Set 1)

  1. Check differences. Constant 1st difference → arithmetic. Constant 2nd difference → quadratic an = An² + Bn + C.
  2. Check ratios. Constant → geometric. Ratios forming 2, 3, 4, 5… → factorial.
  3. Compare to known families: squares (n²), cubes (n³), triangular (n(n+1)/2), factorial (n!), Fibonacci.
  4. Check signs. Alternating → factor of (−1)n or (−1)n+1.
  5. Close to a known family? Compare term-by-term to n², n³, 2n, etc. and note the correction.
  6. Try recursive. If each term depends on the previous term(s) in a simple way.

Two new tactics for Set 2

  1. Coefficient on n² isn't always 1. If 2nd differences equal 2A, the n² coefficient is A. So 2nd diff = 4 → 2n² + … ; 2nd diff = 6 → 3n² + … .
  2. Mixed formulas. Some sequences combine two patterns — e.g. polynomial times exponential, like n · 2n−1. If one lens fails, try two.

Answer Key
Problem 11. Quadratic (n² + n)   an = n² + n = n(n+1)
2nd diffs = 2 → n² coefficient is 1. Subtract n²: 1, 2, 3, 4 = n. (This is 2× the triangular numbers.) a7 = 49 + 7 = 56
Problem 12. Cubes + 1   an = n³ + 1
n³ = 1, 8, 27, 64, 125, 216. Each term is 1 more. Shifted-family trick from Set 1 #1, now with cubes. a7 = 343 + 1 = 344
Problem 13. Factorial (recursive form)   a1 = 1, an = n · an−1
Same sequence as Set 1 #2, now as a recursion. Check: a2=2·1=2, a3=3·2=6, a4=4·6=24 ✓. a7 = 7! = 5040
Problem 14. Quadratic (half coefficient)   an = ½n² + &frac32;n = n(n+3)/2
1st diffs 3, 4, 5, 6, 7. 2nd diffs 1, 1, 1, 1. Since 2A=1, n² coefficient = ½. Subtract ½n²: 1.5, 3, 4.5, 6, 7.5 → &frac32;n. a7 = 24.5 + 10.5 = 35
Problem 15. Alternating squares (opposite parity)   an = (−1)n+1 · n²
Magnitudes are n². Signs start positive at n=1, so use (−1)n+1. Flip of Set 1 #5. a7 = (−1)⁸ · 49 = +49
Problem 16. Lucas numbers (Fibonacci family)   a1 = 2, a2 = 1, an = an−1 + an−2
Same recurrence as Fibonacci, different seeds. 2+1=3, 1+3=4, 3+4=7, 4+7=11, 7+11=18. The Fibonacci rule defines a whole family. a7 = 11 + 18 = 29
Problem 17. Quadratic (n² − 2)   an = n² − 2
n² = 1, 4, 9, 16, 25, 36. Each term is 2 less. 2nd diffs = 2 confirm quadratic. a7 = 49 − 2 = 47
Problem 18. Exponential + 1   an = 2n + 1
2n = 2, 4, 8, 16, 32, 64. Each term is 1 more. Mirror of Set 1 #10. Not geometric. a7 = 128 + 1 = 129
Problem 19. Squared triangular (Nicomachus)   an = [n(n+1)/2]²
√terms: 1, 3, 6, 10, 15, 21 — triangular numbers! So an = (triangularn)². Also equals the sum of first n cubes. a7 = 28² = 784
Problem 20. Polynomial × exponential   an = n · 2n−1
Divide by 2n−1: 1, 2, 3, 4, 5, 6 = n. Product of arithmetic factor (n) and geometric factor (2n−1). a7 = 7 · 64 = 448
What Set 2 reinforces: 11, 14, 17 → quadratic family · 12 → cubes (shifted) · 13 → factorial (recursive) · 15 → alternating signs (opposite parity) · 16 → Fibonacci rule with different seeds · 18 → exponential + 1 · 19 → composed families · 20 → mixed polynomial × exponential.
Diagnostic: If Claire nails 11, 12, 15, 17, 18 easily but struggles with 14, 19, 20 — good sign. Basic families are down; she just needs more composite-pattern exposure. If she struggles with the easy ones too, go back to Set 1 fundamentals (differences and ratios).
