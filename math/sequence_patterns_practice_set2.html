How to use this set: These 10 problems use the same six pattern families from Set 1, but in different forms. The goal is automaticity — when Claire sees a sequence, she should reach for the checklist without thinking.

The Six-Step Checklist (same as Set 1)

  1. Check differences. Constant 1st difference → arithmetic. Constant 2nd difference → quadratic an = An² + Bn + C.
  2. Check ratios. Constant → geometric. Ratios forming 2, 3, 4, 5… → factorial.
  3. Compare to known families: squares (n²), cubes (n³), triangular (n(n+1)/2), factorial (n!), Fibonacci.
  4. Check signs. Alternating → factor of (−1)n or (−1)n+1.
  5. Close to a known family? Compare term-by-term to n², n³, 2n, etc. and note the correction (e.g. n² − 1, 2n + 1).
  6. Try recursive. If each term depends on the previous term(s) in a simple way.

Two new tactics for Set 2

  1. Coefficient on n² isn't always 1. If second differences equal 2A (not just 2), then the n² coefficient is A. So 2nd diff = 4 → 2n² + … ; 2nd diff = 6 → 3n² + … .
  2. Mixed formulas. Some sequences combine two patterns — e.g. a polynomial times an exponential, like n · 2n−1. If one lens alone fails, try building up from two.

Problem 11. Quadratic (n² + n)
Formula: an = n² + n   (equivalently, n(n+1))
First differences: 4, 6, 8, 10, 12. Second differences: 2, 2, 2, 2 — constant, so quadratic with n² coefficient = 1. Subtract n²: 2−1=1, 6−4=2, 12−9=3, 20−16=4. That's just n. So an = n² + n. (This is 2× the triangular numbers — see Set 1 #4.)
a7 = 49 + 7 = 56
Problem 12. Cubes + 1
Formula: an = n³ + 1
Compare: n³ = 1, 8, 27, 64, 125, 216. Each given term is exactly 1 more. Same "shifted family" idea as Set 1 #1, but with cubes instead of squares.
a7 = 343 + 1 = 344
Problem 13. Factorial (recursive form)
Recursive rule: a1 = 1, an = n · an−1
Same sequence as Set 1 #2 (n!), but now expressed as a recursive rule. Check: a2 = 2·1 = 2, a3 = 3·2 = 6, a4 = 4·6 = 24. ✓ This is the definition of factorial — recursive and explicit forms describe the same sequence.
a7 = 7! = 5040
Problem 14. Quadratic
Formula: an = ½n² + &frac32;n   (equivalently, n(n+3)/2)
First differences: 3, 4, 5, 6, 7. Second differences: 1, 1, 1, 1. Since 2A = 1, the n² coefficient is ½. Subtract ½n²: 2−½=1.5, 5−2=3, 9−4.5=4.5, 14−8=6, 20−12.5=7.5. Those grow by 1.5 each time — arithmetic with first term 1.5. So add &frac32;n: an = ½n² + &frac32;n.
a7 = ½(49) + &frac32;(7) = 24.5 + 10.5 = 35
Problem 15. Alternating squares (opposite parity)
Formula: an = (−1)n+1 · n²
Magnitudes are still n². But signs now start positive when n=1, so we need (−1)n+1 instead of (−1)n: (−1)² = +1 ✓. Compare to Set 1 #5, where signs started negative.
a7 = (−1)⁸ · 49 = +49
Problem 16. Lucas numbers (Fibonacci family)
Recursive rule: a1 = 2, a2 = 1, an = an−1 + an−2
Exact same recurrence as Fibonacci, just different starting values. 2+1=3, 1+3=4, 3+4=7, 4+7=11, 7+11=18. ✓ This shows that the "Fibonacci rule" isn't one specific sequence — it's a rule that defines a whole family based on starting values.
a7 = 11 + 18 = 29
Problem 17. Quadratic (n² − 2)
Formula: an = n² − 2
Compare: n² = 1, 4, 9, 16, 25, 36. Each given term is exactly 2 less. Alternatively, first differences 3, 5, 7, 9, 11 and second differences 2, 2, 2, 2 — confirms quadratic with n² coefficient = 1.
a7 = 49 − 2 = 47
Problem 18. Exponential + 1
Formula: an = 2n + 1
Compare to 2n = 2, 4, 8, 16, 32, 64. Each given term is exactly 1 more. Mirror of Set 1 #10 (2n − 1) — same trick, opposite correction. Also not geometric (ratios aren't constant).
a7 = 128 + 1 = 129
Problem 19. Squared triangular (Nicomachus)
Formula: an = [n(n+1)/2]²
Take square roots: √1=1, √9=3, √36=6, √100=10, √225=15, √441=21. Those are the triangular numbers! So an = (triangularn)² = [n(n+1)/2]². This is also the famous identity: sum of first n cubes = [n(n+1)/2]².
a7 = [7·8/2]² = 28² = 784
Problem 20. Polynomial × exponential
Formula: an = n · 2n−1
Divide each term by 2n−1: 1/1=1, 4/2=2, 12/4=3, 32/8=4, 80/16=5, 192/32=6. That leaves 1, 2, 3, 4, 5, 6 — just n! So an = n · 2n−1. This is the hardest type in the set: the formula is a product of an arithmetic factor (n) and a geometric factor (2n−1).
a7 = 7 · 2⁶ = 7 · 64 = 448
