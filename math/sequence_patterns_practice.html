Why these are hard: If a sequence isn't arithmetic (no constant difference) and isn't geometric (no constant ratio), the formula could be almost anything. The trick is to have a short checklist of patterns you always try — in order — before giving up.

Claire's Checklist — Try these IN ORDER

  1. Check differences. First differences constant → arithmetic. If not, check second differences. Constant second differences → an = An² + Bn + C (quadratic).
  2. Check ratios. Constant ratio → geometric. If ratios are 2, 3, 4, 5… (changing in a pattern), the formula likely involves n! (factorials).
  3. Compare to familiar families:
    • Squares: 1, 4, 9, 16, 25 → n²
    • Cubes: 1, 8, 27, 64, 125 → n³
    • Triangular: 1, 3, 6, 10, 15 → n(n+1)/2
    • Factorial: 1, 2, 6, 24, 120 → n!
    • Fibonacci: 1, 1, 2, 3, 5, 8 → each = sum of previous two
  4. Check signs. Alternating + and − means a factor of (−1)n or (−1)n+1.
  5. Try "close to a square/cube." If terms are 2, 5, 10, 17, 26… they're n² + 1. Always compare to n² and n³ and see what you have to add or subtract.
  6. Try recursive. If each term depends on the one(s) before it in a simple way, write an = an−1 + (something) or an = an−1 + an−2.

Answer Key: Each answer shows the type of pattern, the formula, and the key observation that unlocks it.
Problem 1. Quadratic (shifted square)
Formula: an = n² + 1
Compare: n² = 1, 4, 9, 16, 25, 36. Each given term is exactly 1 more. So an = n² + 1.
a7 = 49 + 1 = 50
Problem 2. Factorial
Formula: an = n!
Ratios: 2/1=2, 6/2=3, 24/6=4, 120/24=5. The ratios themselves form 2, 3, 4, 5… — classic factorial signature. 1! = 1, 2! = 2, 3! = 6, 4! = 24.
a7 = 7! = 5040
Problem 3. Fibonacci (recursive)
Recursive rule: a1 = 1, a2 = 1, an = an−1 + an−2
1 + 1 = 2, 1 + 2 = 3, 2 + 3 = 5, 3 + 5 = 8, 5 + 8 = 13. Each term is the sum of the previous two. (An explicit formula exists — Binet's — but it's beyond IM3.)
a7 = 8 + 13 = 21
Problem 4. Triangular numbers
Formula: an = n(n+1)/2
Check: 1(2)/2=1, 2(3)/2=3, 3(4)/2=6, 4(5)/2=10. ✓ First differences grow arithmetically (2, 3, 4, 5…), so second differences are constant — confirming a quadratic formula.
a7 = 7(8)/2 = 28
Problem 5. Alternating squares
Formula: an = (−1)n · n²
|terms| = 1, 4, 9, 16, 25, 36 = n². Signs alternate starting with negative when n=1, so use (−1)n: (−1)¹ = −1 ✓
a7 = (−1)⁷ · 49 = −49
Problem 6. Quadratic (n² − n)
Formula: an = n² − n   (equivalently, n(n−1))
First differences: 2, 4, 6, 8, 10 — not constant. Second differences: 2, 2, 2, 2 — constant! So it's quadratic. Check: 1−1=0, 4−2=2, 9−3=6, 16−4=12, 25−5=20. ✓
a7 = 49 − 7 = 42
Problem 7. Cubes
Formula: an = n³
1³=1, 2³=8, 3³=27, 4³=64, 5³=125. One of the families on the checklist — recognize instantly.
a7 = 343
Problem 8. Quadratic
Formula: an = n² − n + 3
First differences: 2, 4, 6, 8, 10. Second differences: 2, 2, 2, 2 — constant, so quadratic. Since second difference is 2, the n² coefficient is 1 (because 2A = 2). Subtract n² from each term: 3−1=2, 5−4=1, 9−9=0, 15−16=−1, 23−25=−2. That's arithmetic: −n + 3. So an = n² − n + 3.
a7 = 49 − 7 + 3 = 45
Problem 9. Factorial (shifted)
Formula: an = (n+1)!
2 = 2!, 6 = 3!, 24 = 4!, 120 = 5!, 720 = 6!. Index is shifted by 1 compared to problem 2.
a7 = 8! = 40,320
Problem 10. Exponential minus 1 (Mersenne)
Formula: an = 2n − 1
Add 1 to each: 2, 4, 8, 16, 32, 64 — powers of 2! So each term = 2n − 1. Note this is not geometric (ratios are 3, 7/3, 15/7… not constant), but is built from a geometric pattern.
a7 = 128 − 1 = 127
Pattern summary — what Claire should memorize:
• Constant 2nd differences → quadratic (try n² ± something)
• Ratios forming 2, 3, 4, 5… → factorial
• Each term depends on previous two → Fibonacci-type recursive
• Alternating signs → factor of (−1)n
• Terms look like a known family ± something → rewrite as (family) + correction
• Adding 1 gives powers of 2 → 2n − 1
