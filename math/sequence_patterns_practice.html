Strategy
Why these are hard: If a sequence isn't arithmetic (no constant difference) and isn't geometric (no constant ratio), the formula could be almost anything. The trick is a short checklist of patterns to try — in order.

Claire's Checklist — Try these IN ORDER

  1. Check differences. First differences constant → arithmetic. If not, check second differences. Constant 2nd differences → an = An² + Bn + C (quadratic).
  2. Check ratios. Constant ratio → geometric. Ratios forming 2, 3, 4, 5… → formula likely involves n!.
  3. Compare to familiar families: Squares (n²), Cubes (n³), Triangular (n(n+1)/2), Factorial (n!), Fibonacci (each = sum of previous two).
  4. Check signs. Alternating + and − means a factor of (−1)n or (−1)n+1.
  5. Try "close to a square/cube." Always compare to n² and n³ and see what to add or subtract.
  6. Try recursive. an = an−1 + (something) or an = an−1 + an−2.

Answer Key
Problem 1. Quadratic (shifted square)   an = n² + 1
n² = 1, 4, 9, 16, 25, 36. Each term is exactly 1 more. a7 = 49 + 1 = 50
Problem 2. Factorial   an = n!
Ratios: 2/1=2, 6/2=3, 24/6=4, 120/24=5 — classic factorial signature. a7 = 7! = 5040
Problem 3. Fibonacci (recursive)   a1 = 1, a2 = 1, an = an−1 + an−2
1+1=2, 1+2=3, 2+3=5, 3+5=8, 5+8=13. Each term is the sum of the previous two. a7 = 8 + 13 = 21
Problem 4. Triangular numbers   an = n(n+1)/2
1(2)/2=1, 2(3)/2=3, 3(4)/2=6, 4(5)/2=10. ✓ First differences grow arithmetically, so 2nd differences are constant. a7 = 7(8)/2 = 28
Problem 5. Alternating squares   an = (−1)n · n²
|terms| = n². Signs alternate starting negative when n=1, so use (−1)n. a7 = (−1)⁷ · 49 = −49
Problem 6. Quadratic (n² − n)   an = n² − n
1st diffs: 2, 4, 6, 8, 10. 2nd diffs: 2, 2, 2, 2 — constant → quadratic. Check: 1−1=0, 4−2=2, 9−3=6 ✓. a7 = 49 − 7 = 42
Problem 7. Cubes   an = n³
1³=1, 2³=8, 3³=27, 4³=64, 5³=125. Recognize instantly from the checklist. a7 = 343
Problem 8. Quadratic   an = n² − n + 3
2nd diffs constant (=2) → n² coefficient is 1. Subtract n²: 2, 1, 0, −1, −2 — arithmetic = −n + 3. a7 = 49 − 7 + 3 = 45
Problem 9. Factorial (shifted)   an = (n+1)!
2 = 2!, 6 = 3!, 24 = 4!, 120 = 5!, 720 = 6!. Index shifted by 1 vs. Problem 2. a7 = 8! = 40,320
Problem 10. Exponential minus 1 (Mersenne)   an = 2n − 1
Add 1 to each: 2, 4, 8, 16, 32, 64 — powers of 2. So an = 2n − 1. Not geometric (ratios vary). a7 = 128 − 1 = 127
Pattern summary: Constant 2nd differences → quadratic · Ratios forming 2,3,4,5 → factorial · Each term from previous two → Fibonacci-type · Alternating signs → (−1)n · Close to a known family → family + correction · Add 1 gives powers of 2 → 2n − 1.
