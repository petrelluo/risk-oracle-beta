1. What a Chemical Equation Says (§11.1–11.2)

A chemical equation is a shorthand for a reaction. Reactants on the left, products on the right. A skeleton equation is unbalanced — it shows the correct formulas but not the correct amounts.

SymbolMeaningSymbolMeaning
(s)solid→yields, produces
(l)liquid⇌reversible reaction
(g)gasΔ over →heat required
(aq)aqueousMnO2 over →MnO2 is a catalyst (not consumed)

Words → Skeleton → Balanced

  1. Translate names into correct formulas (use charges / polyatomic ions).
  2. Write reactants → products.
  3. Add state symbols if requested.
  4. Balance using coefficients only — never change a subscript.

2. Balancing — Block/Inventory Method (§11.3)

Law of Conservation of Matter: same number of atoms of each element on both sides. Coefficients = relative amounts (atoms, molecules, moles) — not mass.

Strategy

  1. Write the correct skeleton equation.
  2. Draw a vertical line from the arrow; make an atom-inventory table.
  3. Count atoms in order: Metal → Nonmetal → O → H.
  4. Adjust coefficients; treat polyatomic ions as a unit when they appear on both sides.
  5. Reduce to lowest whole-number ratio.

Worked example: __Fe + __O2 → __Fe2O3

ReactantsProducts
Fe1 → 42 → 4
O2 → 63 → 6

Balanced: 4 Fe + 3 O2 → 2 Fe2O3

Hydrocarbon combustion tip: "If the H count is made divisible by 4, it balances 99% of the time." Balance C first, then H, O last.

3. The Six Reaction Types (§11.4–11.9)

TypePatternRecognize By
Combination §11.4A + B → AB≥2 reactants → one product
Decomposition §11.5AB → A + BOne reactant; often needs Δ or electricity
Combustion §11.6Fuel + O2 → oxides; CxHy + O2 → CO2 + H2OBurns in O2
Single-Replacement §11.7A + BC → AC + BLone element + compound; swap similar elements
Double-Replacement §11.9AB + CD → AD + CBTwo aqueous ionic compounds swap partners

3a. Combination / Synthesis

  • Metal + Nonmetal → Ionic compound: 2 Na + Cl2 → 2 NaCl
  • Metal oxide + water → Base: CaO + H2O → Ca(OH)2
  • Nonmetal oxide + water → Oxyacid: CO2 + H2O → H2CO3

3b. Decomposition

  • Binary → elements: 2 HgO → 2 Hg + O2
  • Metal carbonate → metal oxide + CO2: CaCO3 → CaO + CO2
  • Metal chlorate → metal chloride + O2: 2 KClO3 → 2 KCl + 3 O2

3c. Combustion

  • Hydrocarbon: CxHy + O2 → CO2 + H2O
  • Alcohol: 2 CH3OH + 3 O2 → 2 CO2 + 4 H2O
  • Metal in O2: 2 Mg + O2 → 2 MgO

3d. Single-Replacement & Activity Series (§11.7–11.8)

METALS (most reactive top)
Li, K, Ca, Na, Mg, Al, Zn, Fe, Ni, Sn, Pb, H2, Cu, Ag, Pt, Au
Li–Na react with cold H2O. H2 = H in an acid.
HALOGENS (most reactive top)
F2, Cl2, Br2, I2

Rule: reaction occurs only if the lone element is MORE reactive than the similar element in the compound.

  • Mg + 2 HCl → MgCl2 + H2 ✓
  • Ag + HBr → NO REACTION (Ag below H2)
  • Cl2 + 2 NaBr → 2 NaCl + Br2 ✓

3e. Double-Replacement (§11.9)

Two aqueous ionic compounds swap cations. Reaction occurs only if one product is: a precipitate, a molecular liquid (like water / a weak acid), or a gas.

Example: AgNO3(aq) + NaCl(aq) → AgCl(s) + NaNO3(aq) ✓

5. Quick Identification Flowchart

  1. One reactant? → decomposition. Else step 2.
  2. O2 a reactant? → combustion. Else step 3.
  3. Lone element + compound? → single-replacement.
  4. Two aqueous ionic compounds? → double-replacement.
  5. Reactants combine into one product? → combination.

6. Study Checklist

  • ☐ Translate names → formulas (roman numerals, polyatomic ions).
  • ☐ Write a skeleton equation from a word description.
  • ☐ Balance with block-inventory method.
  • ☐ Predict products for all 6 reaction types.
  • ☐ Use activity series for single replacement.
  • ☐ Use precipitate/liquid/gas rule for double replacement.
  • ☐ Identify the reaction type of any balanced equation.
