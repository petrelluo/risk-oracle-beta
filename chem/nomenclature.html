1. Binary Compounds (Suffix: -ide)

For binary compounds (2 elements), the second element always ends in -ide.

TypeNaming ConventionExample
IonicMetal + Nonmetal-ideNaF: Sodium fluoride
CovalentPrefixes for both elementsP₂O₅: Diphosphorus pentoxide
Binary AcidHydro- + root + -ic acidHI: Hydroiodic acid

2. Covalent (Molecular) Naming Prefixes

Covalent compounds (two nonmetals) use Greek prefixes to indicate the number of atoms for both elements. Note: "Mono-" is omitted for the first element only.

PrefixesCountExample
Mono / Di / Tri1 / 2 / 3CO: Carbon monoxide
Tetra / Penta / Hexa4 / 5 / 6N₂O₄: Dinitrogen tetroxide
Hepta / Octa / Deca7 / 8 / 10Cl₂O₇: Dichlorine heptoxide

3. The Polyatomic Ladder (per-, -ate, -ite, hypo-)

Suffixes and prefixes for oxyanions change based on oxygen count relative to the standard -ate ion.

Prefix/SuffixOxygen CountExample (Bromine)
per-...-ate+1 OxygenBrO₄⁻: Perbromate
...-ateStandard IonBrO₃⁻: Bromate
...-ite-1 OxygenBrO₂⁻: Bromite
hypo-...-ite-2 OxygenBrO⁻: Hypobromite

4. Oxyacid Transitions

When oxyanions form acids, prefixes (per/hypo) remain unchanged, but suffixes swap.

Ion SuffixAcid SuffixExample
-ate-ic acidH₂CO₃ (Carbonate) ➔ Carbonic acid
-ite-ous acidHClO₂ (Chlorite) ➔ Chlorous acid
per-...-ateper-...-ic acidHClO₄ (Perchlorate) ➔ Perchloric acid
hypo-...-itehypo-...-ous acidHClO (Hypochlorite) ➔ Hypochlorous acid
